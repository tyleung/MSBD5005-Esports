# MSBD5005-Esports

Website:  
https://msbd5005-esports.appspot.com/

API:  
https://api-dot-msbd5005-esports.appspot.com/

Running the client:  
`cd client/`  
`npm run start` or `yarn start`

Running the server:  
`cd server/`  
`npm run start` or `yarn start`

Deployment:  
`npm run build` or `yarn build`
`git push origin master`
